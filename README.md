# NLP Tutorial

## Installation
Install all dependencies listed in requirements.txt

## Run

(1) greykode - BERT, BiLSTM, Seq2Seq, NNLM, Seq2SeqAttention, CNN, LSTM, RNN, TRANSFORMER, Word2Vec
``` python
cd ./greykode
python bert.py
python nnlm.py
python transformer.py
python bi_lstm.py
python bi_lstm_attention.py
python seq2seq.py
python seq2seq_attention.py
python text_lstm.py
python text_cnn.py
python text_rnn.py
python word2vec_skipgram_softmax.py
cd ../
```

(2) Hierarchical Attention Network
``` python
cd ./han
python train.py
python test.py
cd ../
```

(3) WaveNet
``` python
cd ./wavenet
python demo.py
cd ../
```
